# School Manager

A console-based school management system built with .NET.

## Features

- Student registration and profile viewing
- Lecturer registration and course assignment
- Course management
- Student course enrollment
- Result entry with automatic grade calculation
- Transcript, course performance, and top-student reports
- JSON file persistence for saved data

## Run

```bash
dotnet run
```

## Data Files

The app stores data in these JSON files in the project root:

- `students.json`
- `lecturers.json`
- `courses.json`
- `enrollments.json`
- `results.json`
